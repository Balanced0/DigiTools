# DigiTools

A clean and professional React.js web application that simulates an e-commerce platform for digital tools (software products). Users can browse tools, add them to a cart, view total pricing, and proceed to checkout.

---

## Tech Stack
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify (NPM Package)
- React Icons
- JSON

---

## Features
- **Dynamic Product Listing:**
    Products (Tools) are loaded from JSON and rendered as cards with name, price, features and icons.
- **Shopping Cart System:** 
    Users can add/remove items, see total cost, and view cart count from the cart icon in the navbar.
- **Window Navigation:** 
    Users can switch between product list and cart while keeping state intact.
